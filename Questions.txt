1. Modern C++ & Move Semantics
Move Semantics: Explain rvalue references (T&&) and how std::move helps avoid expensive copies of temporary objects.
Smart Pointers: Discuss the differences between std::unique_ptr, std::shared_ptr, and std::weak_ptr. Be prepared to explain how shared_ptr uses reference counting and how weak_ptr prevents cyclic dependencies.
Lambda Expressions: Explain capture clauses (by value [=] vs. by reference [&]) and when to use mutable lambdas.
Newer Standards: Mention features like auto type inference, nullptr, constexpr, and C++17's std::optional or std::any.

2. Advanced Memory & RAII
Resource Acquisition Is Initialization (RAII): This is a critical "mid-level" topic. Explain how tying resource lifetimes to object lifetimes ensures exception safety and prevents leaks.
